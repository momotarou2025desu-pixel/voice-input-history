最終更新日: 2026-01-08

1. 概要

「音声入力＆編集キーボード」は、Androidのキーボードアプリ（IME）です。本プライバシーポリシーでは、本アプリにおけるデータの取り扱いについて説明します。

2. 収集する情報

本アプリは、氏名・住所・電話番号など、ユーザーを特定できる個人情報を収集しません。

3. 端末内への保存（メモ／履歴／コピー履歴）

機能提供のため、本アプリは以下のデータをユーザーの端末内に保存することがあります：

  • 保存したフレーズ（メモ／スニペット枠）
  • 確定履歴（キーボードから確定したテキスト）
  • コピー履歴（キーボードが有効な間のクリップボード履歴）

これらのデータは端末内に保存され、本アプリのサーバーへアップロードされません。 端末のアプリ情報から「データを消去」する、またはアプリをアンインストールすることで削除できます。

4. パスワード欄について

本アプリは、パスワード入力欄に関するデータを意図的に履歴として保存しないよう配慮しています。 ただし、動作は利用するアプリやAndroidシステムの仕様により異なる場合があります。

5. 音声入力について

音声入力を有効にした場合、端末の音声認識機能（またはユーザーが選択した音声認識サービス）が音声を処理してテキスト化します。 本アプリは、音声データを本アプリのサーバーへアップロードしたり、本アプリ側のサーバーに音声を保存したりしません。

6. 広告・解析（アナリティクス）

本アプリは、広告SDKや解析ツール（アナリティクス）を使用しません。

7. 第三者提供・共有

本アプリは、ユーザー情報を第三者に販売または共有しません。

8. 権限

  • RECORD_AUDIO：音声入力を有効にした場合にのみ使用します。
